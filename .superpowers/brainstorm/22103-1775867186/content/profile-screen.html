Key Differences Summary

Feature Anonymous Registered
Header "Not Signed In" + CTA User avatar, name, email, location
Stats ❌ Not shown ✓ Report count (total, verified, pending)
My Reports ❌ Not available (shows CTA to sign up) ✓ Recent reports list with status
Quick Actions ❌ Not available ✓ Edit profile, Notification settings
Logout ❌ Not shown (already "logged out") ✓ Log out button

Anonymous Profile Strategy

💡 Conversion Opportunity
Every tap on the Profile tab by an anonymous user is a conversion opportunity. Show value proposition and clear CTA.
✅ Always Allow "Continue as Anonymous"
Never block access. User can view map/feed/alerts without account. Profile just shows account benefits.

Registered User Profile Sections

  • Header - Avatar initials, name, email, municipality
  • Quick Stats - Total reports, verified count, pending count
  • My Reports - Last 3 reports with status (tap "See All" for full history)
  • Quick Actions - Edit profile, Notification settings
  • Log Out - Sign out button (returns to anonymous state)

Full Report History (When Tapped "See All")

Shows all reports chronologically with:

  • Incident type & severity (admin-classified)
  • Location (approximate on public view)
  • Status timeline (submitted → verified → dispatched → resolved)
  • Timestamps for each status change
  • Actions: View details, Edit (if pending), Cancel (if pending)
