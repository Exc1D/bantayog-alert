Citizen App Navigation Structure

Which navigation approach best serves citizens during emergencies?

A

5-Tab Bottom Bar (Spec)

📍 Map • 📋 Feed • 🚨 Report • ⚠️ Alerts • 👤 Profile

Pros: Clear separation, quick access to all features, familiar mobile pattern

Cons: More tabs to navigate, Report button competes with other tabs

B

4-Tabs + Floating Report Button

📍 Map • 📋 Feed • ⚠️ Alerts • 👤 Profile + [🚨] floating action button

Pros: Report always visible, emphasizes primary action, fewer tabs

Cons: FAB obscures content, less standard pattern

C

3-Tabs (Consolidated)

📍 Map & Feed (swipeable) • 🚨 Report (center home) • ⚠️ Alerts & 👤 Profile

Pros: Simpler navigation, Report as primary focus

Cons: Map/Feed combined may be confusing, harder to access Profile
