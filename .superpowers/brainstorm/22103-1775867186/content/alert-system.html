Alert Priority Levels

🔴 Emergency Red gradient ✓ Immediate ✓ Strong pattern ✓ Loud alarm
🟡 Warning Yellow/Orange ✓ Yes ✓ Double pulse ✓ Chime
🟢 Advisory Green/Blue ✓ Yes ✗ No ✓ Gentle ping

Alert Sources (Verified Only)

✅ Official Sources Only
Alerts only come from verified municipal accounts (MDRRMO) and national agencies (PAGASA, PHIVOLCS). Citizens cannot post alerts - only reports that admins verify.

Location-Based Filtering

Users only see alerts relevant to their municipality. Daet residents don't need flood warnings for Labo unless it affects them (e.g., highway closure).

// Filter by user's registered municipality
.where('targetMunicipalities', array-contains(user.municipality))
.orderBy('issuedAt', 'desc')

Alert Lifecycle

  • Issued - Created by admin, push notification sent immediately
  • Active - Visible in Alerts tab, push persists for 24h or until expired
  • Expired - Past validUntil date, moved to bottom, marked "Expired"
  • Resolved - Incident resolved, marked "Resolved", shows resolution time
