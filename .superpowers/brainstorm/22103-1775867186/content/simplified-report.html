Why This Works Better

✓ Citizens: Less Cognitive Load

  • No taxonomy decisions during stress
  • Just describe what you see in your own words
  • Faster submission (5-10 seconds)
  • Reduces "I don't know which category this is" friction

✓ Admins: Better Classification

  • See photo + description + location together
  • Have expertise to classify correctly
  • Can add context from follow-up calls
  • Prevents miscategorization from panicked users
