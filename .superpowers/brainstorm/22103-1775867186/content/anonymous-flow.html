Anonymous vs Registered User Flow

How should citizens interact with the app?

ANONYMOUS FIRST
No signup required to report

Anonymous-First (Spec)

Anyone can report immediately. After first report, gently suggest creating an account. Optional: link past reports via phone number.

LOWEST: Friction, highest participation

ACCOUNT REQUIRED
Sign up to use the app

Registration Required

Create account before first report. Phone verification required. Better data quality, traceable reports.

HIGHEST: Data quality, accountability

3 FREE REPORTS
Then sign up to continue

Quota Before Signup

3 anonymous reports allowed. After that, create account to continue. Balances access with accountability.

BALANCED: Try before you commit

Anonymous-First Flow (Recommended)
1. First Report
User taps "Report" → Submits with phone number → Gets Report ID → Done.
No account asked. Takes 10 seconds.
2. Gentle Nudge (After Report #1)
Report submitted!
Create an account to track this report and get status updates. Takes 30 seconds.
User can dismiss → Returns to map → Can report again anytime
3. Account Creation Links Past Reports
When user finally creates account, they enter phone number → System finds all previous reports from that number → Links them to new account. Now they can track everything.
"We found 2 previous reports from this number. They've been linked to your account."

Anonymous User Capabilities

Feature Anonymous Registered
Submit reports ✓ Unlimited ✓ Unlimited
Upload photos ✓ Yes ✓ Yes
View map/feed ✓ Yes ✓ Yes
Receive alerts ✓ Yes (location-based) ✓ Yes (location + preferences)
Track own reports ✗ No (only via Report ID) ✓ Yes (full history)
Edit reports ✗ No ✓ Yes (until verified)
Cancel reports ✗ No ✓ Yes (pending only)

Why Anonymous-First?

✓ Pros

  • Zero friction - Report in 10 seconds, no forms
  • Higher participation - More reports = better response
  • Emergency-ready - No barriers during crisis
  • Privacy - Some users don't want accounts
  • Compliance - Still collect phone for follow-up

✗ Cons

  • Can't track reports - Unless you save Report ID
  • Can't edit later - If you made a mistake
  • Harder to follow up - Admin can't contact if number wrong

Account Creation Trigger

When to suggest account creation:

  • After first report submitted ✓
  • After viewing 3+ incidents
  • When tapping "Profile" tab (if not logged in)
  • Never block - always "Not Now" option
Key insight:
Most users will create account after first successful report. They see value, then invest. Forced signup = dropoff.
