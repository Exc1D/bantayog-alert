Anti-Patterns to Avoid

❌ DON'T: Block features
Never block reporting or viewing map/feed. Users will uninstall. Always "Not Now" or "Continue as Anonymous" option.
❌ DON'T: Show on every screen
Don't nag. Maximum: 1 nudge per session. Remember "Not Now" for 7 days.
✅ DO: Show value first
Trigger account prompts when users try to do something that requires an account. They'll understand WHY they need it.

Account Linking Magic

// When user finally creates account:
const phoneNumber = input.value;
const pastReports = await db.collection('reports')
  .where('contactInfo.phone', phoneNumber)
  .get();

// Magic moment: Link all past reports!
if (pastReports.length > 0) {
  await showConfetti('🎉 Linked 3 past reports!');
  await pastReports.forEach(report => {
    report.userId = user.uid;
    report.linkedAt = new Date();
    db.collection('reports').doc(report.id).update({userId: user.uid});
  });
}
Result: User creates account → System finds all their past reports → Links them to new account → User sees everything in one place. Instant gratification!
