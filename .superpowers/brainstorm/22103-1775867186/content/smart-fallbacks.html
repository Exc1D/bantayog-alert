Key Principles

  • Try auto first - Attempt GPS/camera detection on mount
  • Fail gracefully - If denied/prompt, show manual inputs immediately
  • Never block - User can always submit report, even with zero permissions
  • Queue for offline - Store in IndexedDB, sync when connection returns
  • Clear feedback - Show "Location ready" ✅ or "Select manually" 📝
